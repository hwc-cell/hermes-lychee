# 🍒 Hermes Lychee

基于 [hermes-desktop](https://github.com/fathah/hermes-desktop) 的中文本地化魔改版。开箱即用，小白友好。

## 改动

| 模块 | 说明 |
|---|---|
| 🌏 全中文本地化 | 设置、错误提示、托盘菜单等全面汉化 |
| 🇨🇳 国产大模型 | DeepSeek / 通义千问 / 智谱 GLM / Kimi / MiniMax 开箱即用 |
| ⚡ 一键配置 | 选服务商自动填模型名和上下文窗口，不用查文档 |
| 🍒 荔枝红主题 | 专属暗色主题 + 所有主题名称汉化 |
| 🌐 联网搜索 | Tavily / Brave / Serper / Bing 多服务商选择 |
| 📖 网关接入指南 | 微信、企微、飞书、钉钉、QQ 分步教程 |
| 🔒 关闭自动更新 | 避免上游覆盖魔改内容 |
| 🛡️ Office 3D 保护 | GPU 崩溃不再闪退 |

## 安装

从 [Releases](https://github.com/hwc-cell/hermes-lychee/releases) 下载最新版 `.zip`，解压拖进「应用程序」文件夹。首次打开需右键 → 打开（越过程坞确认）。

## 开发

```bash
git clone git@github.com:hwc-cell/hermes-lychee.git
cd hermes-lychee
npm install
npm run dev
```
